🤖 Mini Pi 로봇 기본 사용법 ​

Mini Pi의 전원은 3단계로 구성되어 있습니다. 로봇 전원을 켜기 위한 단계 :

로봇 전원을 끄기 위한 단계 :

로봇의 전원 관리 및 부팅 ​

  1. 배터리 전원 켜기/끄기

로봇의 배터리는 다음과 같습니다.

전원 버튼을 한 번 누른 뒤 곧바로 길게 누르면 led가 뜨면서 배터리가 켜집니다. 배터리 전원을 끌 때도 같은 방식으로 버튼을 누르면 led가 꺼지면서 배터리가 꺼집니다.

로봇에 배터리를 연결한 후 배터리의 전원을 켜줍니다.

  1. 로봇 본체 전원 켜기/끄기

로봇 뒷 면은 사진과 같이 크기가 다른 두 개의 전원 버튼이 있습니다.

큰 전원버튼은 로봇의 모터, 작은 전원 버튼은 로봇의 내부 컴퓨터 전원입니다.

큰 전원버튼을 한 번 가볍게 눌러 켜고, 작은 전원 버튼을 한 번 길게 눌러 켜줍니다.

lcd 창에서 loading... 문구가 뜨며 컴퓨터가 켜지고 있습니다. 이 때는 로봇을 한 자세로 고정시켜 주세요. 거치대에 올려놓거나, 앉은 상태가 좋습니다.

로봇이 켜지면 아래 사진과 같이 DEFAULT가 뜹니다.

로봇을 끌 때에는 로봇을 켤 때의 역순으로 버튼을 눌러주세요. 이 때도 마찬가지로 로봇이 거치대에 올려진 상태거나 앉은 상태에서 꺼주세요.

  1. 조이패드 끄기

back+B 버튼을 3초간 눌러주세요. 둥근 버튼 테두리의 불빛이 꺼지면 조이패드가 꺼진 것입니다.

조이패드의 버튼 조합이 바뀌었을 때

위 조합들로 로봇을 움직이려 했을 때, 로봇이 움직이지 않는 경우가 있습니다. 또는 안내되지 않은 버튼을 눌렀을 때 로봇이 움직이는 경우도 있습니다. 이 경우에는 조이스틱의 모드가 바뀐 것으로, 원래대로 바꾸어주면 됩니다.

맨 위 둥근 버튼을 3초간 눌러주십시오. 이후 원래 버튼을 눌러 버튼 조합이 원래대로 복귀 되었는지 확인해주세요. 이를 반복하며 조이패드의 버튼 조합을 원상복귀 시킵니다.

로봇의 모드 전환 ​

Mini Pi는 다양한 모드를 지원합니다. 조이패드를 활용하여 모드를 전환할 수 있습니다.

로봇이 거치대에 올려져있거나 앉은 상태에서 진행해주세요.

  1. LT+RT+B를 눌러 INITIAL 모드로 전환시킵니다.
  2. LT+RT+(X or B)를 눌러 변경되는 모드를 확인하세요.
  3. 원하는 모드가 나오면 LT+RT+A를 눌러 모드를 활성화해주세요.

로봇의 Calibration ​

Mini Pi를 장기간 사용하거나 한 자세로 오랫동안 있을 경우, 모터의 기본 값이 바뀔 수 있습니다. 이를 방지하기 위해, 수시로 로봇의 관절을 calibration하는 것을 추천드립니다.

  1. 로봇을 거치대에 올려주세요. 위 사진과 같이 각 관절이 올바르게 위치해있는지 확인해주세요.
  2. 모드를 RST ZERO로 변경해주세요.
  3. RST DONE이 뜰 때 까지 기다립니다. 로봇을 중간에 건드리지 않습니다.
  4. RST DONE이 뜬 후 INITIAL 모드로 넘어가면 calibration이 잘 수행된 것입니다.

RST FAIL이 뜬 경우, 로봇의 관절 모터들이 연결되었는지, 중간에 로봇의 관절이 건드려지지 않았는지 확인해주세요

VS Code 사용하기 ​

단순한 메모장과 달리, 개발에 특화된 기능들을 무료로 제공하기 때문에 현재 전 세계 개발자들이 가장 많이 사용하는 도구입니다. 구체적인 사용 목적과 장점은 다음과 같습니다.

  1. 생산성 향상 (IntelliSense) 자동 완성: 코드를 칠 때 명령어나 변수 이름을 미리 예측해서 추천해 줍니다. 오타를 줄이고 타이핑 속도를 획기적으로 높여줍니다. 문법 강조: 코드의 색깔을 다르게 표시해 주어 가독성을 높이고 구조를 파악하기 쉽게 해줍니다.
  2. 강력한 확장성 (Extensions) 레고 같은 맞춤 설정: VS Code는 기본적으로 가볍지만, '확장 프로그램(Extensions)'을 설치하여 기능을 무한대로 늘릴 수 있습니다. 예: Python 개발을 한다면 Python 확장을, 웹 개발을 한다면 Live Server를 설치하는 식입니다. 이 덕분에 거의 모든 프로그래밍 언어(C, Java, Python, JavaScript, HTML/CSS 등)를 하나의 에디터에서 다룰 수 있습니다.
  3. 통합된 개발 환경 터미널 내장: 코드를 짜다가 실행하거나 명령어를 입력할 때 별도의 창(CMD, Terminal)을 켤 필요 없이 VS Code 안에서 해결할 수 있습니다. 디버깅(Debugging): 코드의 오류를 찾고 수정하는 과정을 프로그램 내부에서 직관적으로 수행할 수 있습니다. Git 연동: 소스 코드 버전 관리 도구인 Git이 내장되어 있어, 코드 저장 및 협업 관리가 매우 쉽습니다.
  4. 가벼움과 호환성 가벼운 구동: Visual Studio(보라색 아이콘) 같은 무거운 통합개발환경(IDE)에 비해 훨씬 가볍고 실행 속도가 빠릅니다. 크로스 플랫폼: Windows, macOS, Linux 등 어떤 운영체제에서도 똑같이 사용할 수 있습니다.
  5. 무료 및 오픈 소스 마이크로소프트가 개발했지만 완전 무료이며, 지속적인 업데이트와 방대한 커뮤니티 지원을 받을 수 있습니다.

설치하기 ​

Visual Studio Code(VS Code) 설치는 매우 간단합니다. 사용하시는 운영체제(Windows, Mac)에 맞춰 아래 순서를 따라 진행해 보세요.

1. 공식 사이트 접속 ​

가장 먼저 공식 홈페이지에서 설치 파일을 다운로드해야 합니다.

  • 다운로드 링크: https://code.visualstudio.com/
  • 사이트에 들어가면 파란색 [Download for Windows] (또는 Mac) 버튼이 보입니다. 클릭해서 설치 파일을 받으세요.

2. 운영체제별 설치 방법 ​

🖥️ Windows (윈도우) ​
  1. 다운로드한 파일(VSCodeUserSetup...exe)을 실행합니다.
  2. 사용권 계약에 '동의합니다'를 선택하고 [다음]을 누릅니다.
  3. 설치 경로를 확인하고 [다음]을 계속 누릅니다.
  4. ★중요: 추가 작업 선택 화면
    • 이 단계에서 아래 항목들을 모두 체크(V) 하는 것을 강력 추천합니다.
      • ✅ 바탕 화면에 아이콘 만들기 (선택 사항)
      • ✅ Windows 탐색기 파일의 상황에 맞는 메뉴에 "Code(으)로 열기" 작업 추가 (필수 추천)
      • ✅ Windows 탐색기 디렉터리의 상황에 맞는 메뉴에 "Code(으)로 열기" 작업 추가 (필수 추천)
      • ✅ PATH에 추가 (필수)
    • 팁: 'Code로 열기'를 체크하면, 나중에 폴더에 우클릭 한 번으로 바로 코딩을 시작할 수 있어 매우 편리합니다.
  5. [설치] 버튼을 눌러 완료합니다.
🍎 macOS (맥) ​
  1. 다운로드한 파일(.zip)의 압축을 풉니다.
  2. 'Visual Studio Code'라는 응용 프로그램(아이콘)이 나옵니다.
  3. 이 아이콘을 드래그해서 [응용 프로그램(Applications)] 폴더로 옮겨주세요.
  4. 이제 응용 프로그램 폴더나 Launchpad에서 실행하면 됩니다.

3. 설치 후 필수 설정 (한국어 패치) ​

처음 설치하면 메뉴가 전부 영어로 되어 있습니다. 한국어로 바꾸고 싶다면 아래 단계를 따라 하세요.

  1. VS Code를 실행합니다.
  2. 왼쪽 메뉴바에서 테트리스 블록 모양 아이콘 (Extensions, 단축키: Ctrl+Shift+X 또는 맥 Cmd+Shift+X)을 클릭합니다.
  3. 검색창에 Korean이라고 입력합니다.
  4. Korean Language Pack for Visual Studio Code (지구본 모양 아이콘)를 찾아 [Install] 버튼을 누릅니다.
  5. 오른쪽 하단에 "Change Language and Restart"라는 알림창이 뜨면 [Restart]를 클릭합니다.
  6. 프로그램이 재시작되면서 메뉴가 한글로 바뀝니다.

이제 설치가 완료되었습니다! 코딩을 시작하시면 됩니다.

Remote - SSH 확장 프로그램 설치 ​

  1. VS Code를 실행합니다.
  2. 왼쪽 메뉴바에서 확장(Extensions) 아이콘(테트리스 블록 모양)을 클릭합니다.
  3. 검색창에 Remote - SSH를 입력합니다.
  4. 제작사가 Microsoft로 되어 있는 항목을 찾아 [설치(Install)] 버튼을 누릅니다.
    • 참고: 설치 후 왼쪽 메뉴바에 '모니터 모양' 아이콘(원격 탐색기)이 새로 생길 수 있습니다.

서버(호스트) 추가 및 연결하기 ​

가장 쉬운 방법은 왼쪽 하단의 녹색 아이콘을 사용하는 것입니다.

  1. 명령 실행:
    • VS Code 왼쪽 맨 아래 모서리에 있는 녹색 아이콘 >< 을 클릭합니다.
    • 나타나는 메뉴에서 [호스트에 연결... (Connect to Host...)]을 선택합니다.
  1. 새 호스트 추가:
    • 메뉴에서 [새 SSH 호스트 추가... (Add New SSH Host...)]를 클릭합니다.
    • 입력창에 접속하려는 서버의 SSH 명령어를 입력하고 엔터를 칩니다.
    • 형식: ssh 사용자명@서버IP주소
    • 예시: ssh ubuntu@13.124.55.10
  1. 설정 파일 저장:
    • SSH 설정 파일을 어디에 저장할지 묻습니다. 보통 가장 위에 뜨는 기본 경로(예: C:\Users\사용자\.ssh\config)를 선택하면 됩니다.
    • 오른쪽 하단에 "호스트가 추가되었습니다"라는 알림이 뜨면 [연결(Connect)] 버튼을 누릅니다.

접속 완료 및 확인 ​

  1. 새로운 VS Code 창이 열리면서 연결을 시도합니다.
  2. 플랫폼 선택: 상단 중앙에 "Linux", "Windows", "macOS" 중 서버의 운영체제를 고르라고 나옵니다. (대부분의 서버는 Linux입니다.)
  3. 비밀번호 입력: (SSH 키를 설정하지 않았다면) 서버의 비밀번호를 입력하라는 창이 뜹니다. 입력 후 엔터를 칩니다.
  4. 연결 확인:
    • 왼쪽 하단 녹색 아이콘 옆에 SSH: 서버IP 라고 표시되면 연결 성공입니다.
    • 이제 왼쪽 탐색기(Ctrl+Shift+E)에서 [폴더 열기]를 누르면 내 컴퓨터가 아닌 원격 서버의 폴더가 보입니다.

💡 꿀팁: 비밀번호 없이 접속하기 (SSH Key 설정)

매번 비밀번호를 치는 것이 귀찮다면, 로컬 컴퓨터에서 SSH 키를 생성하여 서버에 등록하는 것이 좋습니다. 로컬 터미널(CMD/PowerShell)에서 ssh-keygen -t rsa 입력하여 키 생성. 생성된 공개키(id_rsa.pub)의 내용을 복사. 서버의 ~/.ssh/authorized_keys 파일 안에 붙여넣기. 이렇게 하면 다음부터는 비밀번호 없이 바로 VS Code가 열립니다.
